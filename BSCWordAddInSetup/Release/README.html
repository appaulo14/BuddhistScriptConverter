Useful Links

NOTE: Below are supplementary files, fonts, and software that enhance the usefulness of my software. They are not included in Buddhist Script Converter due to licensing issues.

  1. Pali Roman Keyboard Input:
  1. Allows the user to easily enter Pali Roman characters without using the Character Map. For example, ctrl + alt + a gives ā.
  2. http://fsnow.com/pali/keyboard/
  1. Fonts:
  1. Serif Font similar to Times New Roman for Pali in Roman transliteration: http://sourceforge.net/projects/linuxlibertine/
  2. Devanagari Font with good conjunct support: http://tdil.mit.gov.in/download/GISTYogeshN.htm
  3. Khmer Mūla Font: http://www.cambodia.org/fonts/ (Download KhmerUnicode2.0.0.zip on this page)
